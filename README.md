# Seeker Dash

**Localized On-chain Activity & Fee Tracker for Solana Seeker.**

Seeker Dash is a professional, privacy-first dashboard designed exclusively for the Solana Seeker mobile phone. It provides real-time tracking of on-chain activities and cumulative gas fees for your Seed Vault wallet.

## Features
- **Real-time Monitoring**: Track every transactionpayer activity via Helius WSS.
- **28-Day History**: Visual grid display of historical on-chain engagement.
- **Fee Tracking**: Accurate calculation of SOL spent on transaction fees.
- **100% Privacy**: No servers, no tracking, all data stays on your Seeker device.

---

## Privacy Policy / 隐私政策

### English Version
1. **Data Collection**: Seeker Dash DOES NOT collect, store, or transmit any personal information, private keys, or seed phrases. 
2. **Architecture**: 100% client-side. All data queries are made directly from the user's device to the Solana blockchain via Helius RPC.
3. **Local Storage**: All tracking data is stored exclusively on the user’s local device. 
4. **No Third-Parties**: We do not integrate any third-party analytics or tracking SDKs.
5. **Security**: Users are solely responsible for their wallet security. This is a read-only monitoring tool.
